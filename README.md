# Mis Scripts Personales

## Descripción

Este repositorio contiene una colección de scripts que he desarrollado para facilitar diversas tareas cotidianas en mi entorno de trabajo. Estos scripts pueden ser de utilidad para otros usuarios, pero se proporcionan "tal cual" sin garantías de ningún tipo. Si decides utilizarlos, hazlo bajo tu propia responsabilidad.

## Contenido

La carpeta `scripts` contiene todos mis scripts organizados por categoría o funcionalidad. Cada script tiene su propia documentación en el archivo correspondiente.

## Uso

1. Clona este repositorio en tu máquina local:
   ```bash
   git clone https://github.com/tu_usuario/my_scripts.git

2. Navega hasta el directorio del repositorio:
   ```bash
    cd mis_scripts

3. Explora los scripts disponibles y léete la documentación de cada uno para entender cómo usarlos.

## Requisitos

    Linux (basado en Debian preferiblemente)
    Bash/zsh
    Otros requisitos específicos se documentan en cada script
